# Prologue
어느 마을에 한 전사가 살고 있었습니다. 그 전사는 모든 것이 완벽했지만, 단 하나 가지고 싶은 것이 있었습니다. 바로, 다이아몬드였습니다. 그의 마을에서 찾을 수 없었던 다이아몬드는 그의 마음을 움직이게 했고, 결국 전사는 위험을 무릅쓰고 다이아몬드를 찾으러 가기로 결심했습니다.
과연.. 전사는 다이아몬드를 손에 넣을 수 있을까요?
